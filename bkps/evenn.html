SEE 2082 - Computer Science (Even Days)

Set 1: Advanced Programming

1 What is polymorphism in OOP?
Answer: Polymorphism is the ability of an object to take on many forms. The most common use is when a parent class reference is used to refer to a child class object.
2 Differentiate between compiler and interpreter.
Answer:
Compiler: Translates entire program at once, generates executable file
Interpreter: Translates line by line, no executable file generated

Set 2: Web Technology

3 What is CSS and what are its types?
Answer: CSS (Cascading Style Sheets) is used for styling web pages.
Types:
1. Inline CSS
2. Internal CSS
3. External CSS

Set 3: Computer Fundamentals

4 What are the generations of computer?
Answer:
1. First Generation (1940-1956): Vacuum Tubes
2. Second Generation (1956-1963): Transistors
3. Third Generation (1964-1971): Integrated Circuits
4. Fourth Generation (1971-Present): Microprocessors
5. Fifth Generation (Present-Beyond): Artificial Intelligence

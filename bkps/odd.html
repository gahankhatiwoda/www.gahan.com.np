SEE 2082 - Computer Science (Odd Days)

Set 1: Programming Fundamentals

1 What is the output of following C program?
#include <stdio.h>
int main() {
    int x = 5;
    printf("%d", x++ * ++x);
    return 0;
}
A) 25
B) 30
C) 36
D) Undefined behavior
Answer: D) Undefined behavior
Explanation: The expression x++ * ++x has undefined behavior because x is modified multiple times between sequence points.
2 Which data structure uses LIFO principle?
A) Queue
B) Stack
C) Linked List
D) Tree
Answer: B) Stack
Stack follows Last-In-First-Out (LIFO) principle.

Set 2: Database Management

3 What is the full form of SQL?
A) Structured Query Language
B) Simple Query Language
C) Standard Query Language
D) System Query Language
Answer: A) Structured Query Language
SQL stands for Structured Query Language used for managing relational databases.
4 Which SQL command is used to remove a table?
A) DELETE
B) DROP
C) REMOVE
D) TRUNCATE
Answer: B) DROP
DROP TABLE command removes the table structure and data permanently.

Set 3: Networking

5 What does HTTP stand for?
A) Hyper Text Transfer Protocol
B) High Text Transfer Process
C) Hyper Transfer Text Protocol
D) High Transfer Text Process
Answer: A) Hyper Text Transfer Protocol
HTTP is the foundation of data communication for the World Wide Web.
